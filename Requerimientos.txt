--Requerimientos
    --REQ-F-1: El sistema debe analizar un string y decir si la cantidad de signos (), {}, [] estan balanceadas, o sea, que para cada
                signo abierto, haya uno cerrado.
    --REQ-NF-1: El sistema debe responder al usuario en un tiempo menor a dos segundos el resultado del analisis.
    --REQ-NF-2: El sistema debe responder al usuario en un tiempo menor a dos segundos el resultado de las pruebas al analisis.

--Criterios de Exito
    -CE-REQ-F-1: El sistema retorna de manera correcta el resultado del examen basado en el string suministrado.
    -CE-REQ-NF-1: El sistema retorna la respuesta en menos de dos segundos.
    -CE-REQ-NF-2: El sistema retorna los resultados de las pruebas en menos de dos segundos.

--Escenarios de prueba
    --"()" => true
    --"{[)" => false
    --"a+(c+b)" => true
    --"({[]})" => true
    --"}" => false
